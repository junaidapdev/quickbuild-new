## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
```

### Production

```bash
npm start
```

## Design Highlights

- **Color Scheme**: Deep blue (#1E40AF) primary, electric cyan (#06B6D4) accent
- **Typography**: Inter and Outfit font families
- **Animations**: Fade-in on scroll, hover effects, parallax, gradient animations
- **Effects**: Glassmorphism, gradient backgrounds, smooth transitions

## Sections

1. **Hero** - Full viewport with animated gradient and code editor mockup
2. **Process** - Three-step process with numbered cards
3. **Features** - 2x3 grid of feature cards with glassmorphism
4. **Portfolio** - Horizontal scrolling carousel
5. **Pricing** - Two pricing tiers
6. **Testimonials** - Rotating client testimonials
7. **FAQ** - Accordion-style questions
8. **CTA** - Final call-to-action
9. **Footer** - Links and information
